About Sloyd

Sloyd offers parametric AI 3D generation, creating customizable game-ready models from text prompts. Unlike other tools that generate fixed meshes, Sloyd creates parametric models with adjustable properties, allowing users to fine-tune dimensions, proportions, and details after generation. This approach produces cleaner, more controllable 3D assets ideal for game development and product design.

Key Features

  • Parametric 3D model generation
  • Adjustable properties post-generation
  • Text to 3D conversion
  • Game-ready mesh output
  • Dimension and proportion control
  • Clean mesh topology
  • Standard format exports
  • API for integration

Pros

  • Unique parametric approach
  • Adjustable models after generation
  • Clean topology for games
  • Good for iterative design
  • Free tier available

Cons

  • Limited to certain model types
  • Less variety than Meshy
  • Parametric constraints can be limiting
