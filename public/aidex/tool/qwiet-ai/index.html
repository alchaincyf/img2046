About Qwiet AI (by Harness)

Qwiet AI (formerly ShiftLeft) is an AI-powered code security platform that uses Code Property Graph (CPG) technology for deep vulnerability detection. Acquired by Harness in September 2025, it is now integrated into the Harness DevSecOps platform. The CPG technology enables advanced reachability analysis that reduces open-source vulnerability noise by 92% and accelerates remediation by up to 95%. The platform includes SAST, secrets detection, SCA, and security training for developers. It achieves a 97% true positive rate with a 90% reduction in false positives.

Key Features

  • Code Property Graph (CPG) technology for deep analysis
  • Advanced reachability analysis reducing noise by 92%
  • SAST with 97% true positive rate
  • Intelligent SCA with exploitability context
  • Secrets detection in source code
  • Developer security training integration
  • Integration with Harness CI/CD platform

Pros

  • Extremely high accuracy (97% true positive rate)
  • CPG technology provides unique depth of analysis
  • Dramatically reduces false positives (90% reduction)
  • Now backed by Harness with broader DevSecOps integration
  • Fast remediation with guided fix recommendations

Cons

  • Acquired by Harness; standalone product future uncertain
  • Pricing less transparent after acquisition
  • Smaller community compared to Snyk or Checkmarx
  • Integration mainly optimized for Harness ecosystem
