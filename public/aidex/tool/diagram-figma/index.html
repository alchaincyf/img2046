About Diagram

Diagram is a design tools company focused on building AI-powered tools for designers. They created the popular Magician Figma plugin and Genius (AI-powered auto-complete for Figma components). Diagram's products aim to augment designer capabilities with AI while keeping designers in control of the creative process.

Key Features

  • AI-powered Figma plugins
  • Auto-complete for Figma components
  • Icon and image generation in Figma
  • Design system intelligence
  • Workflow automation tools
  • Regular new product releases
