About NVIDIA ACE

NVIDIA ACE (Avatar Cloud Engine) provides a suite of AI technologies for creating lifelike digital characters in games. The platform includes Audio2Face for automatic lip-sync animation from audio, Riva for speech recognition and synthesis, and NeMo for language understanding. Game developers use ACE to give NPCs natural speech, facial expressions, and conversational abilities. NVIDIA partners with studios like Convai and Inworld for production-ready implementations.

Key Features

  • Audio2Face automatic lip-sync
  • Riva speech recognition and synthesis
  • NeMo language understanding
  • Real-time facial animation
  • Cloud and local deployment
  • Game engine integration
  • Partner ecosystem (Convai, Inworld)
  • Multi-language support
