About Blooket

Blooket is a gamified quiz platform that combines question sets with various unique game modes like Tower Defense, Gold Quest, and Cafe. Teachers create or import question sets, then students play through different game formats that keep review sessions fresh and exciting. Its variety of game modes makes it highly replayable.

Key Features

  • Unique game modes
  • Question set import
  • Tower Defense mode
  • Gold Quest mode
  • Homework assignments
  • Student analytics

Pros

  • Very engaging game modes
  • Kids love variety
  • Affordable premium
  • Easy to get started

Cons

  • Games can overshadow learning
  • Limited question types
  • Some modes require Plus
