About WaveFunctionCollapse

WaveFunctionCollapse (WFC) is a constraint-solving algorithm for procedural content generation that generates tile maps, 3D models, and other structured content by observing a small example and generating larger, structurally similar outputs. Originally developed by Maxim Gumin, WFC has become popular in game development for generating level layouts, dungeon maps, building interiors, and terrain patterns that maintain local coherence while being globally novel. Multiple game engines have WFC implementations (Unity plugins, Unreal marketplace assets), and it forms the mathematical basis of modern AI map generators.

Key Features

  • Procedural tile map generation from small examples
  • Constraint-based generation ensuring local coherence
  • 3D voxel and 2D tile variants
  • Unity and Godot plugin implementations available
  • Deterministic generation with seed support
  • Customizable tile adjacency rules
  • Infinite map generation with consistent style
  • Multiple pattern synthesis algorithms

Pros

  • Free and open-source algorithm
  • Generates novel content that matches artistic style of examples
  • Many ready-made plugins for major game engines
  • Well-understood algorithm with extensive documentation

Cons

  • Algorithm can fail to find solutions (backtracking required)
  • Setup requires curating good example tiles
  • Not AI-driven—deterministic constraint solver
  • Limited to structural patterns without gameplay context
