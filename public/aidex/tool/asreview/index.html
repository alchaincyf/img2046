About ASReview

ASReview LAB is an open-source, browser-based software for AI-aided systematic screening of textual data, developed at Utrecht University. It uses active learning to interactively prioritize records using AI models that learn from your labeling decisions, reducing workload by up to 95%. Version 2.0 enables efficient, transparent AI-assisted systematic reviews by combining multiple machine learning models with input from expert crowds. Users can switch between fast, general-purpose, and domain-specific models (Ultra, Multilingual, Heavy) within the same project. ASReview LAB Server supports crowd screening, allowing multiple experts to screen a single project using the same AI model.

Key Features

  • Active learning for intelligent screening prioritization
  • Pre-configured AI models (Ultra, Multilingual, Heavy)
  • Crowd screening with ASReview LAB Server
  • Customizable tags and groups for data extraction
  • Dark mode, keyboard shortcuts, font size customization
  • 24.1% performance improvement in v2 vs v1
  • No data collection or tracking

Pros

  • Completely free and open source
  • 95% workload reduction through active learning
  • Strong privacy (no data collection)
  • Multiple AI model options for different use cases
  • Active academic community and ongoing development

Cons

  • Requires Python 3.10+ and some technical setup
  • Less polished UI compared to commercial tools
  • No built-in reference management or PDF viewing
  • Limited collaboration features (v2 adds crowd screening)
  • No official support team; relies on community
