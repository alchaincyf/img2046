About Relume

Relume is an AI-powered website design tool that helps designers create websites faster. Enter your business type and description, and Relume's AI generates a complete sitemap, wireframes for each page, and can export to Figma or Webflow. It includes a comprehensive design system library with 1,000+ Figma and Webflow components. Relume is especially popular among Webflow designers for its seamless workflow.

Key Features

  • AI sitemap generation from business description
  • AI wireframe creation for each page
  • 1,000+ Figma and Webflow component library
  • Copy generation for all website sections
  • Export directly to Figma or Webflow
  • Continuous updates to component library

Pros

  • Dramatically speeds up website design workflow
  • Excellent Figma and Webflow integration
  • Large high-quality component library
  • AI copywriting included

Cons

  • Subscription cost adds up
  • Learning curve for component library
  • Best value only for Webflow users
  • Component customization can be limited
