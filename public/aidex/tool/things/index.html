About Things

Things is a beautifully designed task management app exclusively for Apple platforms (macOS, iPadOS, iOS, watchOS, and visionOS). Created by Cultured Code in Stuttgart, Germany, Things 3 is considered the most well-rounded, best designed, and easiest-to-use GTD app available. The app features a clean, intuitive interface with powerful organization tools that help users manage tasks and projects efficiently. Recent updates include Things Cloud with Fastlane that pushes newly set reminders to all devices instantly, and a new Slim Mode that allows users to collapse the sidebar to cut out distractions and focus on the task at hand. Things excels at task management (90% of reviewers use it primarily for this) with features like projects, areas, checklists, tags, and a Today view.

Key Features

  • Beautiful, intuitive design
  • Things Cloud with Fastlane sync
  • Slim Mode for focus
  • Projects and areas
  • Checklists and tags
  • Today view
