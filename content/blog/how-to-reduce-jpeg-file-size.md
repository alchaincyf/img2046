# 如何减小JPEG文件大小而不损失质量 | 图像魔方

JPEG（Joint Photographic Experts Group）是全球最流行的图片格式之一。据统计,互联网上超过70%的图片是JPEG格式。从手机拍摄的照片到网站的产品图,JPEG无处不在。但JPEG文件通常较大,占用存储空间,影响网页加载速度。如何在减小JPEG文件大小的同时保持良好的画质,成为了每个人都需要掌握的技能。本文将深入探讨JPEG压缩的原理和最优设置。

## JPEG压缩原理

### JPEG是如何工作的？

**JPEG压缩流程**：
```
原始图像
    ↓
1. 色彩空间转换（RGB → YCbCr）
    ↓
2. 色彩采样（Chroma Subsampling）
    ↓
3. 分块处理（8×8像素块）
    ↓
4. 离散余弦变换（DCT）
    ↓
5. 量化（Quantization）← 这里控制质量
    ↓
6. 熵编码（Huffman编码）
    ↓
压缩后的JPEG
```

### 关键步骤详解

**1. 色彩空间转换**

**RGB → YCbCr**：
```
RGB（红绿蓝）：
- 适合屏幕显示
- 三个通道相关信息多
- 冗余数据多

YCbCr（亮度、蓝色色度、红色色度）：
- Y（Luma）：亮度，人眼敏感
- Cb（Chroma Blue）：蓝色色度
- Cr（Chroma Red）：红色色度
- 适合压缩
```

**为什么要转换？**
```
人眼特性：
- 对亮度变化敏感
- 对色度变化不敏感

转换优势：
- 可以减少色度数据
- 保持视觉质量
- 大幅减小文件
```

**2. 色彩采样（Chroma Subsampling）**

**原理**：
```
保留完整亮度（Y）
减少色度数据（Cb, Cr）

常见采样格式：
- 4:4:4：无采样（文件最大）
- 4:2:2：色度减半
- 4:2:0：色度1/4（最常用）
```

**4:2:0采样详解**：
```
每个2×2像素块：
- 亮度（Y）：4个值（完整保留）
- 色度（Cb, Cr）：各1个值（共享）

数据量：
- 原始：4×3 = 12个值
- 4:2:0：4+1+1 = 6个值
- 减少：50%

视觉影响：
- 几乎无影响
- 人眼难以察觉
- 文件减小30-40%
```

**3. 离散余弦变换（DCT）**

**作用**：
```
将图像从空间域转换到频率域：
- 空间域：像素的明暗值
- 频率域：变化的速度和方向

低频 = 平缓变化（天空、墙面）
高频 = 快速变化（边缘、纹理）
```

**为什么这样做？**
```
人眼特性：
- 对低频信息敏感
- 对高频信息不敏感

压缩策略：
- 保留低频信息
- 舍弃或减少高频信息
- 减小数据量
- 保持视觉质量
```

**4. 量化（Quantization）**

**核心步骤**：
```
质量参数在这里起作用！

量化表：
- 预定义的数值表
- 决定每个频率成分的精度
- 质量高 → 量化步长小 → 精度高
- 质量低 → 量化步长大 → 精度低
```

**质量参数的作用**：
```
质量100（最高）：
- 几乎不量化
- 文件最大
- 接近无损

质量90：
- 轻微量化
- 文件减小10-20%
- 质量几乎无损

质量80：
- 中等量化
- 文件减小30-40%
- 视觉质量优秀（推荐）

质量70：
- 较大量化
- 文件减小50-60%
- 仔细观察有轻微损失

质量60及以下：
- 大幅量化
- 文件减小70%+
- 明显质量损失
```

**5. 熵编码（Huffman编码）**

**作用**：
```
无损压缩步骤：
- 统计数据出现频率
- 常见数据用短编码
- 罕见数据用长编码
- 进一步减小文件（10-20%）
```


### 技巧4：检查压缩伪影

**常见伪影识别**：

**1. 蚊子噪点（Mosquito Noise）**
```
位置：高对比边缘
特征：边缘周围的噪点
原因：高频成分量化过度

解决：
- 提高质量参数
- 或使用智能压缩
```

**2. 色块（Color Blocking）**
```
位置：平滑区域（天空、墙面）
特征：方块状的色块
原因：DCT块可见

解决：
- 提高质量参数到80+
- 或应用轻微模糊
```

**3. 环形伪影（Ringing）**
```
位置：锐利边缘周围
特征：光环或环状伪影
原因：高频量化导致

解决：
- 提高质量参数
- 或锐化前压缩
```

**4. 色带（Banding）**
```
位置：渐变区域
特征：可见的色带
原因：渐变精度不足

解决：
- 提高质量参数
- 或添加轻微噪点
```

**检查方法**：
```
1. 100%缩放查看
2. 检查高对比边缘
3. 检查平滑区域
4. 检查渐变区域
5. 放大200%检查细节
```

### 技巧5：避免重复压缩

**问题**：
```
JPEG有损压缩：
每次编辑保存都会损失质量

原始（100%）
  ↓ 编辑保存90%
压缩后1（90%）
  ↓ 再次编辑保存85%
压缩后2（85%×90%=76.5%）
  ↓ 再次编辑保存80%
压缩后3（76.5%×85%=65%）

质量逐渐下降！
```

**最佳实践**：
```
✅ 工作流程：

原始文件（RAW/TIFF）
  ↓
编辑（无损格式）
  ↓
导出最终版本（JPEG质量80）

❌ 避免：

原始JPEG
  ↓ 编辑保存JPEG
  ↓ 再次编辑保存JPEG
  ↓ 继续编辑保存JPEG

✅ 如需多次编辑：
1. 从原始文件开始
2. 或转换为PNG/TIFF编辑
3. 最后导出JPEG
```
