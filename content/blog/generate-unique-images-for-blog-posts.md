- 配图影响SEO排名(搜索引擎优先展示有图片的内容)

### 2. SEO和搜索引擎优化

Google等搜索引擎重视多媒体内容:
- **图片SEO**: 图片文件名、ALT文本影响排名

- **分享链接**: 配图增加社交媒体分享,获得外链

**最佳实践**:

### 3. 品牌识别和差异化

在内容饱和的互联网环境中:

## 免费图库的局限性

### 问题1: 千篇一律,缺乏个性

**现实情况**:
- Unsplash同一张热门图片被数千个网站使用
- 读者在不同平台反复看到相同图片
- 你的内容看起来"大众化"
- 难以建立视觉识别度

**案例**:
搜索"business meeting"的图库图片:
- 至少10,000+网站使用前10张图片
- 读者立即识别出"这是图库图片"
- 降低内容的专业性和独特性感知

### 问题2: 难以完美匹配主题

**图库图片的局限**:
- **预设场景**: 图库摄影师预设的场景,可能不完全匹配你的内容
- **通用性**: 为了适用多个场景,图库图片往往过于通用
- **缺乏细节**: 无法包含你文章中的特定概念、关键词、品牌元素
- **风格限制**: 被动接受图库摄影师的风格,而非自己的风格

**示例**:
写一篇关于"AI在电商中的应用"的文章:
- 图库搜索"AI ecommerce": 结果泛泛,缺乏针对性
- 生成图片: 可以精确包含"AI算法、电商数据、智能推荐"等元素

### 问题3: 版权和许可问题

**免费图库的陷阱**:
- **许可限制**: 部分图库禁止商业使用
- **署名要求**: 部分要求署名,影响设计

## 博客配图最佳实践

### 1. 配图位置和数量

**文章结构配图建议**:
- **头图(Featured Image)**: 文章顶部,1张,必需
- **章节分隔**: 每500-800字1张图片,可选
- **概念解释**: 复杂概念配图说明
- **总结配图**: 文章结尾,可选

**WordPress设置**:
- 特色图片尺寸: 1200 x 630像素
- 内容图片: 宽度1200像素,高度自适应
- 对齐: 居中或左对齐
- 间距: 上下各20px

### 2. 图片尺寸规范

**不同平台要求**:

**博客平台**:
- WordPress特色图片: 1200 x 630像素
- Medium: 1000 x 500像素
- Ghost: 2000 x 1200像素

**社交媒体分享**:
- Facebook: 1200 x 630像素(1.91:1)
- Twitter: 1200 x 675像素(16:9)
- LinkedIn: 1200 x 627像素(1.91:1)
- Pinterest: 1000 x 1500像素(2:3竖版)

**推荐尺寸**:
- 生成1600 x 900像素,适应多平台
- 使用[裁剪工具](/crop)快速调整为不同平台尺寸

### 3. 配色和品牌一致性

**建立配色方案**:
1. 选择2-3种主色调
2. 定义辅助色
3. 在所有配图中使用相同配色
4. 在提示词中指定颜色

**示例配色**:
```
科技博客配色:
- 主色: #0066CC(蓝色)
- 辅色: #00CCFF(青色)
- 背景: #F5F5F5(浅灰)

提示词: "蓝色和青色渐变,科技背景..."
```

### 4. 文字和Logo叠加

**使用图像魔方[文字卡片工具](/text-card-generator)**:
1. 生成AI图片后下载
2. 上传到文字卡片工具
3. 添加文章标题或关键词

- 文字简洁(3-8个字)

**建立系列视觉识别**:
1. 创建模板(相同布局、配色)
2. 批量生成配图
3. 只更换主体元素
4. 保持固定元素位置(如Logo、标题)

