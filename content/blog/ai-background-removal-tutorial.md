## 高级技巧：获得最佳效果

### 1. 拍摄技巧：好的输入才有好的输出

虽然AI可以处理各种图片，但遵循以下拍摄技巧能获得最佳效果：

**光线设置**：
```
✅ 推荐：
- 均匀的漫射光（避免强烈阴影）
- 柔光箱或自然光
- 主体光线充足

❌ 避免：
- 强烈逆光
- 复杂多重阴影
- 主体过暗或过曝
```

**背景选择**：
```
✅ 推荐：
- 纯色背景（白墙、背景布）
- 与主体对比度高的颜色
- 简单场景

❌ 避免：
- 背景与主体颜色相近
- 过于杂乱的背景
- 背景中有其他物体
```

**拍摄角度**：
```
✅ 推荐：
- 正面或稍侧角度
- 完整显示主体
- 避免遮挡

❌ 避免：
- 极端角度
- 主体被遮挡
- 边缘模糊不清
```

## 常见问题与解决方案

### Q1：为什么我的图片处理后边缘不自然？

**可能原因**：
1. 原图分辨率太低（<500px）
2. 原图主体边缘模糊
3. 主体与背景对比度低

**解决方案**：
```
优化方案：
1. 使用高分辨率原图（建议>1000px）
2. 拍摄时确保焦点清晰
3. 提高主体与背景对比度

后期修复：
1. 下载AI处理结果
2. 使用"滤镜工具"轻微模糊边缘（0.3-0.5px）
3. 或使用自由画布手动修正
```

### Q2：透明背景PNG文件很大怎么办？

**原因**：PNG格式保留所有透明信息，文件较大

**解决方案**：
```
方法1：压缩PNG
使用"图片压缩"工具：
- 上传透明PNG
- 质量设为80-90
- 文件减小30-50%，质量损失很小

方法2：转换为JPG（如不需要透明背景）
1. 添加白色/其他背景
2. 导出为JPG
3. 文件减小80%+

方法3：使用WebP格式
1. 导出为WebP + 透明
2. 比PNG小30-50%
3. 质量相同
```

### Q3：批量处理时部分图片效果不好？

**原因**：图片质量参差不齐

**解决方案**：
```
策略1：分批处理
- 按质量分组：高质量批次、低质量批次
- 对低质量批次单独处理或人工干预

策略2：先优化再处理
- 统一调整尺寸和对比度
- 再进行背景移除

策略3：人工复核
- 批量处理后抽查
- 发现问题图片单独重新处理
- 或手动微调
```

### Q4：如何保留阴影？

**挑战**：AI通常会移除所有背景，包括地面阴影

**解决方案**：
```
方法1：后期添加阴影
1. 完成背景移除
2. 在自由画布中添加：
   - 椭圆形阴影图层
   - 高斯模糊（10-20px）
   - 降低透明度（30-50%）
   - 放在物体图层下方

方法2：保留原图阴影（手动）
1. 生成两个版本：
   - 版本A：AI背景移除（透明）
   - 版本B：原图（含阴影）
2. 在自由画布中叠加
3. 用蒙版保留阴影部分
```

### Q5：处理后的图片有锯齿怎么办？

**原因**：
- 原图质量低
- 放大查看时明显
- AI算法偶尔产生

**解决方案**：
```
快速修复：
1. 使用"滤镜工具"
2. 选择"边缘平滑"或轻微模糊
3. 导出优化版本

高级修复（自由画布）：
1. 添加轻微高斯模糊图层（0.5px）
2. 使用"锐化"工具（Amount: 50%）
3. 导出最终版本
```
